PAINT.NET Instructions

    With painting/model:
        Use "Unit template"
        Cut model background
        Add model in new layer over template
        Scale model to template using "Bicubic" filtering/quality
        Apply Object Shadow to model with Shadow Radius 3 Shadow Strength 0.75 and RGB 137
        Merge Layers

    With B&W photo:
        Colorize photo (use one of many online Deep Learning services) (https://hotpot.ai/colorize-picture)
        Use "Unit template"
        Clear template expect frame
        Add photo in new layer under template and position it
        Merge Layers
